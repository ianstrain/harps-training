# HRAPS Training Schedule

A mobile-friendly training schedule for coaches and players.

## Sessions

- **Duration:** February 3rd - March 24th, 2026
- **Frequency:** Weekly (Tuesdays)
- **Location:** The Aura
- **Time:** 7:30 PM - 8:30 PM

*Note: No training on March 17th (St. Patrick's Day)*

## Session Format

Each 1-hour session includes:
- Warm-up (10 mins)
- Drills (20 mins)
- Match (30 mins)

## Features

- Mobile-optimized design
- Edit mode to update drill content
- Auto-saves changes locally

## Deployment

### Option 1: Netlify (Recommended)

1. Go to [app.netlify.com](https://app.netlify.com)
2. Click "Add new site" → "Import an existing project"
3. Connect your GitHub account
4. Select this repository
5. Click "Deploy"

### Option 2: GitHub Pages

1. Go to repository Settings → Pages
2. Select "main" branch as source
3. Save - site will be live at `https://ianstrain.github.io/hraps-training`

## Editing Drills

1. Open the schedule page
2. Toggle "Edit Mode" ON
3. Click in any drill text box and type your content
4. Changes save automatically when you click away
